"# Learnagram" 
DESCRIPTION:
  Learngram is a social media like platform aimed to improve the communication, interactions  between teachers and students and also amoung students.It is where users share educational content that helps student community.

PROBLEM STATEMENT:
  Student often spend time on socialmedia focused on entertainment. Learnagram transforms it into educational micro-learning experiences.

FEATURES
  1.Role bases access control(Students/Teachers)
  2.Posts among peer community
  3.Like and comments system
  4.Follow Friend and Proffessors
  5.Save Posts
  6.Messages and Classroom creation

TECH STACK
  1.FrontEnd: React.js, CSS
  2.Backend: (planned)Node.js, Express
  3.Database: (planned) MongoDB
  4.Authentication: (planned)JWT
  5.Deployment:  (planned)Vercel
